import { Component } from '@angular/core'; import { CommonModule } from '@angular/common'; import { FormsModule } from '@angular/forms'; import { Router, RouterLink } from '@angular/router'; import { AuthService } from '../../../services/auth.service'; @Component({ selector: 'app-signup', standalone: true, imports: [CommonModule, FormsModule, RouterLink], templateUrl: './signup.component.html', styleUrls: ['./signup.component.scss'] }) export class SignupComponent { userId: string = ''; username: string = ''; password: string = ''; inviteKey: string = ''; errorMessage: string = ''; successMessage: string = ''; isLoading: boolean = false; constructor( private authService: AuthService, private router: Router ) {} onSignup(): void { if (!this.userId || !this.username || !this.password || !this.inviteKey) { this.errorMessage = 'Please fill in all fields'; return; } this.isLoading = true; this.errorMessage = ''; this.successMessage = ''; this.authService.signup(this.userId, this.username, this.password, this.inviteKey).subscribe({ next: (response) => { console.log('Signup successful:', response); this.successMessage = 'Signup successful! Redirecting to login...'; setTimeout(() => { this.router.navigate(['/login']); }, 2000); }, error: (error) => { console.error('Signup failed:', error); this.errorMessage = error.error || 'Signup failed. Please try again.'; this.isLoading = false; }, complete: () => { this.isLoading = false; } }); } }
